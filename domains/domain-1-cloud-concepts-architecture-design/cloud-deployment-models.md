# Cloud Deployment Models

Priority: 🟡 Important  
Exam Frequency: ⭐⭐⭐⭐

## Public Cloud

Cloud services are delivered over shared provider infrastructure and available to multiple customers.

Examples:

- Azure
- AWS
- Google Cloud

## Private Cloud

Cloud capabilities are dedicated to one organization. The private cloud may be hosted on-premises or by a third party.

## Hybrid Cloud

Hybrid cloud combines private and public cloud environments with integration between them.

Common use cases:

- Gradual migration
- Data residency
- Legacy application integration
- Disaster recovery

## Community Cloud

Cloud infrastructure shared by organizations with common requirements, such as government, healthcare, or research groups.

## Exam Tip

If the question emphasizes regulatory needs, shared mission, or similar compliance obligations among multiple organizations, consider community cloud.
