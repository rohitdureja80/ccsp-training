# Cloud Roles and Responsibilities

**Domain:** 1 - Cloud Concepts, Architecture & Design  
**Priority:** 🔴 Critical  
**Exam Frequency:** ⭐⭐⭐⭐⭐

## Overview

Cloud security responsibilities are divided across several roles. CCSP questions often test whether you can identify who is responsible for a specific activity.

## Core Cloud Roles

| Role | Primary Responsibility |
|---|---|
| Cloud Customer | Uses cloud services and remains accountable for its data, identities, configurations, and legal obligations. |
| Cloud Service Provider (CSP) | Provides cloud services and manages provider-owned infrastructure and services. |
| Cloud Partner | Supports either the customer or CSP through integration, implementation, consulting, or managed services. |
| Cloud Broker | Intermediates, aggregates, arbitrages, or helps manage cloud services between customers and providers. |
| Cloud Carrier | Provides network connectivity and transport between customer and cloud provider. |
| Cloud Auditor | Independently assesses cloud services, controls, compliance, and security posture. |

## CCSP Exam Tips

- If the question asks who is **ultimately accountable for data**, think **cloud customer**.
- If the question asks who provides **independent assurance**, think **cloud auditor**.
- If the question asks who provides **connectivity**, think **cloud carrier**.
- If the question asks who helps select, aggregate, or manage multiple cloud services, think **cloud broker**.
