# Cloud Computing Concepts

Priority: 🔴 Critical  
Exam Frequency: ⭐⭐⭐⭐⭐

## Definition

Cloud computing provides on-demand access to shared computing resources such as compute, storage, networking, databases, and applications over a network.

## Essential Characteristics

Common cloud characteristics include:

- On-demand self-service
- Broad network access
- Resource pooling
- Rapid elasticity
- Measured service

## Why This Matters for CCSP

Cloud environments are dynamic, scalable, API-driven, and shared. Security must account for:

- Multi-tenancy
- Shared responsibility
- Elastic resources
- Automation
- Identity-based access
- Continuous monitoring

## Traditional vs Cloud Thinking

| Traditional Infrastructure | Cloud Infrastructure |
|---|---|
| Static servers | Dynamic resources |
| Manual provisioning | API-driven provisioning |
| Physical ownership | Shared provider/customer responsibility |
| Network perimeter focus | Identity and policy focus |
| Capital expense | Operational expense |

## Exam Tip

If a question emphasizes elasticity, automation, resource pooling, or self-service, think cloud architecture fundamentals.
