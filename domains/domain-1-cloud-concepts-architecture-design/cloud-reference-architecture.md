# Cloud Reference Architecture

Priority: 🟡 Important  
Exam Frequency: ⭐⭐⭐⭐

## Overview

A cloud reference architecture describes the major roles, components, and relationships in a cloud environment.

## Common Cloud Roles

- Cloud Customer: Uses cloud services
- Cloud Service Provider: Provides cloud services
- Cloud Service Partner: Supports cloud services, such as brokers or auditors
- Cloud Auditor: Performs independent assessments
- Cloud Carrier: Provides network connectivity

## Why It Matters

The reference architecture helps clarify responsibilities, control ownership, trust boundaries, and where security controls apply.

## Security Architecture Concerns

- Identity and access management
- Network segmentation
- Data protection
- Logging and monitoring
- Compliance responsibilities
- Tenant isolation
