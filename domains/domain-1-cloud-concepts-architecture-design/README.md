# Domain 1 - Cloud Concepts, Architecture & Design

## Overview

Domain 1 focuses on understanding cloud computing concepts, architectures, security principles, and design considerations.

The goal is not only to understand cloud technologies, but to understand how security decisions change when moving from traditional infrastructure to cloud-native environments.

- Service models

- Cloud Reference Architecture

### Security Architecture

- Identity-centric security
- Federation

### Cloud Design
- Secure cloud design principles
- Resilience patterns
- Availability patterns
- Portability
- Interoperability
