# Domain 1 - Cloud Concepts, Architecture & Design

## Overview

Domain 1 focuses on cloud computing concepts, cloud architecture, design principles, security architecture, and the business/security implications of using cloud services.

The key exam skill is understanding how cloud changes security decision-making compared to traditional infrastructure.

## Topics

- Cloud computing concepts
- Cloud service models
- Cloud deployment models
- Shared Responsibility Model
- Cloud reference architecture
- Zero Trust Architecture
- Identity and federation
- Cloud design patterns
- Interoperability and portability
- High-yield Domain 1 review
